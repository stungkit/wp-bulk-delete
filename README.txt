== Description ==

WP Bulk Delete allows you to delete and clean anything like posts, comments, users, meta fields, taxonomy terms in bulk. with powerful filter options. 

**Here are some features of this plugins.**

* Delete Posts.
* Delete Posts by posttype.
* Delete Posts by poststatus.
* Delete Posts by date interval.
* Delete Posts by Taxonomy.
* Delete Posts by authors.
* Delete Comments.
* Delete Comments by status.
* Delete Comments by date interval.
* Delete Users.
* Delete Users by userroles.
* Delete Users by date interval.
* Delete Meta fields.
* Delete Post meta fields.
* Delete User meta fields.
* Delete Comment meta fields.
* Delete Taxonomy terms.
* Cleanup ( Auto Drafts, Trash and Revisions)
* Cleanup ( Orphaned Meta and Duplicated Meta )

**Additional [PRO Version](https://xylusthemes.com/plugins/wp-bulk-delete/) Features**

* Delete Posts by title or content.
* Delete Posts by custom fields.
* Delete Posts by any combination.
* Delete Comments by comment author.
* Delete Comments by comment post.
* Delete Users by meta fields.
* More filter on bulk deletion
* Assign deleted user data to another user.

== Installation ==

**This plugin can be installed directly from your site.**

1. Log in and navigate to Plugins & Add New.
2. Type "WP Bulk Delete" into the Search input and click the "Search" button.
3. Locate the "WP Bulk Delete" in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.

**It can also be installed manually.**

1. Download the "WP Bulk Delete" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "WP Bulk Delete" in the list and click the "Activate" link.
